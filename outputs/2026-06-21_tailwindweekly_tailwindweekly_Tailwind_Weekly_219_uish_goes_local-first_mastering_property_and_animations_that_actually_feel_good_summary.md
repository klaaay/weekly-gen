### [优秀动画与卓越动画](https://emilkowal.ski/ui/good-vs-great-animations?ref=tailwindweekly.com)

**原文标题**: [Good vs Great Animations](https://emilkowal.ski/ui/good-vs-great-animations?ref=tailwindweekly.com)

以下是您提供的文章内容的摘要：

动画课程报名已开启，但现已截止。本文由设计工程师 Emil Kowalski 撰写，旨在分享从“好”动画进阶到“优秀”动画的实用技巧。

- 🎯 **注重动画的起源**：动画应自然地从触发元素（如按钮）的位置开始，而非凭空出现。可使用 CSS 的 `transform-origin` 属性（如 `bottom center`）来调整，Radix 和 shadcn/ui 等工具可简化此过程。
- ⚙️ **选择正确的缓动函数**：缓动是动画的核心。对于屏幕上已有的元素移动，应使用 `ease-in-out` 曲线来模拟自然的加速和减速。在多数情况下，推荐默认使用 `ease-out` 曲线。
- 🎨 **使用自定义缓动曲线**：CSS 内置的缓动曲线通常效果不够理想。除基础的悬停效果外，建议使用自定义曲线以获得更佳动感。推荐使用 `easing.dev` 或 `easings.co` 等网站来生成曲线。
- 🔄 **实现基于弹簧的交互**：将 UI 变化直接与鼠标位置绑定会显得生硬。推荐使用 Framer Motion（现称 Motion）的 `useSpring` 钩子，通过弹簧效果让交互更自然。注意，此方法仅适用于装饰性动画，功能性动画（如银行应用中的图表）则不宜使用。
- 🛠️ **精通你的工具**：了解在何种场景下使用哪些 CSS 属性至关重要。例如，实现标签切换动画时，使用 `clip-path` 而非直接改变颜色，能让颜色过渡与高亮条的移动更协调。熟悉工具还能激发创意，如利用 3D 变换制作轨道旋转或硬币翻转等效果。
- 💡 **动画的重要性**：在软件功能趋同的今天，出色的动画能让产品脱颖而出，提升用户对产品的感知和体验。
